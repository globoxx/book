Basic Sudoku Resolver

Due to some issues with the solver, it can currently only solve easy/medium sudokus.

If at a point the resolver can't resolve what you told him to resolve, consider what you gave him too hard.

How the Solver Works

The solver uses logical Sudoku techniques. It never guesses — it only places a number when it is guaranteed to be correct.

1. Single Candidate (Lone Cell)

If a cell can only contain one number, that number is placed.

Example: A cell can only be 7 → the solver places 7.

2. Lone Number in a Row

If a number can only appear in one cell of a row, it is placed there.

Example: In a row, the number 5 fits in only one cell → 5 is placed.

3. Lone Number in a Column

If a number can only appear in one cell of a column, it is placed there.

4. Lone Number in a Box

If a number can only appear in one cell of a box, it is placed there.

5. Naked Pairs

If two cells in the same row, column, or box contain the same two possible numbers, then:

  • Those two numbers must go in those two cells
  • They are removed from all other cells in that group

Example: Two cells contain only {3, 8} → other cells can no longer use 3 or 8.

6. Pointing Pairs

If all possible positions for a number inside a box are:

  • in the same row, or
  • in the same column

then that number can be removed from the rest of that row or column outside the box.

7. Validation Rules

The solver checks for errors before and after solving:

  • No duplicate numbers are allowed in a row
  • No duplicate numbers are allowed in a column
  • No duplicate numbers are allowed in a box

If an invalid number or a contradiction is detected, solving is stopped.
