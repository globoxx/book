Security

We use Python's Cryptography library for secure encryption.

We generate SHA256 keys, store data encrypted on the server, and provide robust protection against unauthorized access.

At SafeMarks, we take security seriously. Our application uses the Python Cryptography library to encrypt all user data. We employ SHA-256 to generate secure encryption keys and AES-256 (Advanced Encryption Standard) for encrypting your grade data. This ensures that your data is both protected and stored securely on our servers.

Additionally, the encrypted data is saved in a text file on the server, allowing users to manage it easily while maintaining strict data privacy.
