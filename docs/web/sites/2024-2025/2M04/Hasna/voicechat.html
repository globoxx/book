Comment obtenir le voice chat ?

Index

Pour obtenir le voice chat sur Roblox, voici les étapes à suivre :

1. Vérifiez votre âge

Conditions : Vous devez avoir au moins 13 ans pour accéder au voice chat.

Vérifiez que votre compte Roblox a une date de naissance correcte.

2. Activez la vérification de l'âge

Allez dans les Paramètres de votre compte (icône en forme de rouage).

Cliquez sur l'onglet Informations sur le compte.

Cherchez l'option Vérification de l'âge et lancez le processus.

Préparez une pièce d'identité valide (comme une carte d'identité, un passeport, ou un permis de conduire).

Suivez les instructions pour télécharger une photo de votre pièce d'identité et une selfie en direct pour vérification.

3. Activez le voice chat

Une fois votre âge vérifié, retournez dans Paramètres.

Cliquez sur l'onglet Confidentialité.

Activez l'option Activer la communication vocale (Voice Chat).

4. Utilisez un micro

Assurez-vous d'avoir un microphone fonctionnel connecté à votre appareil.

Vérifiez que Roblox peut accéder à votre micro via les paramètres de votre appareil.

5. Accédez à des expériences compatibles

Le voice chat est disponible uniquement dans les jeux où les développeurs ont activé cette fonctionnalité. Cherchez l'icône d'un micro dans ces jeux.

Si vous avez des difficultés,voici une vidéo youtube qui vous aidera à comprendre plus facilement la démarche.
